# Hospital Management Appointment System

 Overview
The Hospital Management Appointment System is a web-based application designed to automate hospital appointment scheduling and patient management. It helps patients book appointments online, doctors manage schedules, and administrators maintain records efficiently.

 Features
- Patient Registration and Login
- Doctor Management
- Appointment Booking and Cancellation
- Appointment Scheduling
- Patient Record Management
- User-Friendly Interface
- Database Integration

 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

 Project Objective
The main objective of this project is to simplify the manual hospital appointment process and improve efficiency in managing patient and doctor records.

Installation
1. Clone the repository
2. Move the project folder to the server directory (`htdocs` in XAMPP)
3. Import the database file into phpMyAdmin
4. Start Apache and MySQL
5. Open the project in your browser
